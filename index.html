1. HTML (HyperText Markup Language)

,HTML não é linguagem de programação, mas sim de marcação. É um arquivo de texto que tem delimitadores (tags) que são interpretados pelos navegadores. Cada tag tem um nome e uma função, que reproduz um comportamento específico

2. Estrutura básica do HTML

  • É uma boa prática que a pasta principal do site seja denominada de index.html
  • A maioria das tags precisam ser abertas (< >) e fechadas (< / >).
  • No corpo do editor devemos abrir e fechar a tag html. Ela vai indicar para o navegador onde nossa página começa e termina, além de indicar o que ele "vai ler".
  • Depois abrimos as tags head e body:
    • no head colocamos as configurações de comportamento
    • no body colocamos as informações que serão exibidas para os usuários

Nosso arquivo ficou assim:

< html >

< head >

< /head >

< body >

< /body >

< /html >

3. Falando sobre tags

Aqui veremos alguns exemplos de tags

Como dito antes, as tags que mudam o comportamento da página devem ser escritas no head. Um exemplo é a tag title, que muda o nome da aba do navegador

< title >Página principal< /title >

As tags com conteúdo que será exibido para os usuários são escritas no body, como a i (deixa o texto itálico) e strong (deixa o texto em negrito). Essas tags podem ser usadas em conjunto, exemplo:

< strong > < i > Meu primeiro texto < /i > < /strong >

Fica assim:

Meu primeiro texto

< strong > Meu primeiro < i > texto < /i > < /strong >

Fica assim:

Meu primeiro texto

4. Atributos de tags

Atributos são propriedades que algumas tags podem ter. Existem atributos genéricos e específicos.

Exemplo de atributos genéricos:

id - é um identificador de elementos. O JavaScript tem o poder de ir no html, pegar um elemento específico e trocar o conteúdo dele. Pense assim, você tem vários parágrafos < p > mas quer mudar o conteúdo de 2 deles. Você atribui o mesmo id para nomeá-los e direcionar o JavaScript com base nesse id

< p id = "destaque" > O poder da programação < /p >

class - faz o mesmo que o id, mas para o CSS

< p class = "paragrafo-para-destacar" > O poder da programação < /p >

style - serve para colocar comandos de formatação do texto

< p style = "color:blue;" > O poder da programação < /p >

Fica assim:

O poder da programação

Exemplo de atributos específicos:

input type = "_____" - ela adiciona a página uma caixa de texto. Já o type é um atributo específico da tag input, que aponta qual o tipo de conteúdo aquela caixa aceitará. Exemplo: < input type = " number " > só aceitará número (lembrando que algumas letras podem ser aceitas por causa do algarismo romano)

img src = "_____" - img é uma tag para adicionar imagem. O caminho para vincular essa imagem é com o atributo src. Nele, podemos colocar uma imagem direta de outro site (colocando o endereço dela entre as aspas) ou subir a imagem para o projeto e colocando o caminho (nome da pasta + arquivo, por ex.) entre as aspas.

5. Algumas Tags para texto (tipografia)

  • As tags h's - identificam os títulos. Vai de h1 até h6 e seu número é atribuído de acordo com sua relevância (onde h1 é atribuído ao título principal).
  • < p > - tag para parágrafos. Usada para textos simples.
  • < blockquote > - tag para citações. Essas tags destoa o texto da tag p, o que dá destaque ao que for escrito nela (que ainda pode ser editada com o CSS).
  • < u > - deixa o que for escrito entre ela sublinhada.
  • < small > - adiciona um texto pequeno no que for escrito entre ela.
  • < mark > - deixa o que for escrito entre ela com o aspecto de marca texto.
  • < hr > - adiciona uma linha onde é adicionada que ajuda a separar o conteúdo.
  • < sup > - acrescenta um texto ou letra escrita na parte superior. Ex; Erika@ (@ adicionado com a tag sup).

6. Listas

Existem 2 tipos de listas, ordenadas (< ol >) e não ordenadas (< ul >). Essas são tags pai, que precisam de uma tag filho (nesse caso, < li >) para "funcionar". Elas são muito usadas pois, além de listas, podemos aplicá-las para criar menus no site (que também podem ser editadas em CSS).

Exemplo de ol

< ol >

< li > Começo < /li >

< li > Meio < /li >

< li > Fim < /li >

< /ol >

No navegador, fica assim:

  1. Começo
  2. Meio
  3. Fim

Nesse exemplo, a ordem é dada pelo número na frente dos fatores.

Exemplo de ul

< ul >

< li > Começo < /li >

< li > Meio < /li >

< li > Fim < /li >

< /ul >

No navegador, fica assim:

  • Começo
  • Meio
  • Fim

Nesse exemplo, os números são trocados por "pontos".

7. Links

Link vai além da url de um site. É também ligar uma página a outra dentro de um mesmo site. Exemplo:

< a href = "https://www.dio.me/" title = "Clique e confira" target = "_blank" > Dio.me< /a >

Vamos decifrar essa linha:

  • A tag a vem de anchor (âncora). Ela faz exatamente isso, "ancora a página em um ponto" e, para ir para outro, você pode clicar em um link;
  • O href="_____" é onde colocamos o endereço para onde queremos encaminhar o usuário (no exemplo, o site da DIO);
  • O title adiciona um balão com o texto "Clique e confira" quando o usuário passar o mouse por cima do link;
  • Target informa para o navegador se, ao clicar no link, a página vai se sobrepor a atual (_self) ou ser aberta em uma nova guia (_blank);

Essa linha fica assim:

Dio.me

Agora, pense em um site de uma empresa que tem várias páginas (Sobre, Contato, Inicial, Clientes,...) que, por ser interna, posso fazer o link de outra forma. No editor de códigos, cada página tem seu arquivo, que são denominados:

about.html - contact.html - start.html - client.html

Para fazer o link é só adicionar ao < a href = "____" >. Ex:

< a href = "about.html" > Sobre< /a >

Outra situação é uma página única, com todo o conteúdo do site que, ao clicar no hiperlink nada será carregado mas a página descerá/subirá para o conteúdo. Exemplo:

< h1 > A história da internet < /h1 >

< h2 > Índice < /h2 >

< ul >

< li > < a href = "#surgimento" > Surgimento da internet < /a >< /li >

< li > < a href = "#como-funciona" > Como funciona o HTML? < /a >< /li >

< li > < a href = "#tags" > O que são TAGS? < /a >< /li >

< /ul >

< h2 id = "surgimento" > Surgimento da internet < /h2 >

< h2 id = "como-funciona" > Como funciona o HTML? < /h2 >

< h2 id = "tags" > O que são TAGS? < /h2 >

Ao colocar a # no href criei um id. Dentro dos textos em h2, coloquei novamente os id's para os quais o usuário será direcionado (posso fazer o mesmo no índice), ou adicionar uma opção voltar com um href para o início.
